# ScamAway-Docs
Documentation for the [ScamAway API](https://github.com/ScamAway-API)

## Setup

```sh
git clone https://github.com/Xenorio/ScamAway-Docs
cd ScamAway-Docs
pip install -r requirements.txt
mkdocs serve
```

## Building

```sh
mdocs build
```
